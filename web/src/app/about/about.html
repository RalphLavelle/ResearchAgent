We use AI!

In order to find the events it lists here this website uses a large language model (LLM) in a couple of ways. LLMs are one type of AI, of which ChatGPT is probably the most famous. So how does it work?

How the agent uses AI

Every hour or so, we use an LLM to come up with variations on the question ‘What musical events are on in the next 30 days in Brisbane and the Gold Coast?’ Being an incredibly expensive piece of software the LLM actually knows about these locations and can come up with variations of the locations and the words ‘music events’, like ‘gigs’, ‘What’s on in Fortitude Valley?’, ‘concerts’, etc. The queries it generates never quite repeat themselves so over time the search results the app gets back finds a very broad range of websites. Yes you could do this without an LLM but it would be like playing Voodoo Chile on a ukulele.

Second, as anyone who has ever tried parsing, or ‘scraping’, a website can tell you, that’s fraught with fragility and frustration. Ordinary websites, such as the ones that have gig listings, are not particularly set up to be easily read by machines. That’s where the LLM comes in again: it’s good at turning messy data into data you can use. When the search terms we generated find results that look promising we send the pages to the LLM which picks out individual gigs (and venues, images, dates, etc.) when it finds them. If you didn’t have an LLM you’d better have a large budget and a lot of patience.

Lastly, a lot of gigs show up on more than one web page, especially for the big bands. The problem of duplicates is a well-known one in this situation. ‘Powderfinger live at the Tivoli’ is obviously the same gig as ‘The Tivoli Theatre, Fortitude Valley, proudly presents…Powderfinger!’. Once again, our trusty LLM comes to the rescue and decides that there are not in fact two bands, both called Powderfinger, playing in the same theatre on the same night. Which is a shame, of course, but that’s just how it is.

So there you have it. AI in the verse, and AI in the chorus. At the end of the day, though, the only thing that matters is that we find all the gigs on in Brisbane and the Gold Coast that are out there on the web.
