For task 4, a change was made to have the event link, when clicked, show a small UI with more events in the database for that same act/event. That seems to be working ok in cases where there actually are same/similar events, e.g. 'Open Mic Night'. The problem is that when there are NO similar events, it defaults to showing the first 8 or so events in the database.

# Fix
Where no similar events happen, don't link the event title - it shouldn't even be clickable.
