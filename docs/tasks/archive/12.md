# Make the images more specific
At the moment, many events which share the same url are ending up with the same generic image. This is bad - Photos are very important to this whole project. Ideally, each event should have its own thumbnail image.

# Improve the curator step
The normalize step should try to extract a photo that matches the event for its compilation of the resources, not just show the generic 'events page' one. This may be via a caption on the picture, alt text, text near the picture, etc.

Maybe this is not the best spot to do this, or maybe there are several, but in any case, I want distinct, individual images for events.
