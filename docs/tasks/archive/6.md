# Missing events
Not all concerts within the data range are being found on webpages that the app seems to have found.

## Description
The app searches for events/gigs then finds pages that have information about gigs on. However, even when it successfully finds pages that have events it seems not to get **all** the events that pages has.

## Example
The database has found an event called 'The Buzz Lovers Perform The Best of Nirvana' and links it to this web page: https://allevents.in/miami-au/concerts
But this webpage has other events that fall within the dates we're interested in. For instance, the Baggy Trousers gig, and a few more. Why is the agent only finding the Buzz Lovers one? It should find every event for the time span, not just one.

## Task
Update the code that is causing the agent to miss events on pages it has found so that it gets ALL musical events within the interval (one month) we're interested in.
