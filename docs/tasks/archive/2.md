# Restructuring of the output data from the agent
Instead of outputting a HTML table (from the database) I want the data to generate a JSON file that the frontend can use directly.

## Steps
* Have the agent generate this json file at the point where it had been generating a HTML table. THe json file should have a top-level property called 'generated' with the generated time, and the 'events'. 
* Save it into a suitable location in the frontend.
* Change the frontend to use that JSON to generate its own HTML rather than (in the past) simply embedding the pre-generated HTML
* Any request to the component showing the list of events will result in rendered HTML showing the latest list
