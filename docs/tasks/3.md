# Targeted search
NOrmally, searchterms are derived by the agent using prompt guides, etc., but occasionally I'd like to do a 'targeted' search. Sometimes I know there's a band that I want to have in the listings, etc. and I want to start an agent run with just a single search.

## Admin search panel
Make a search panel in the admin home page that takes a search phrase and starts a full run but with just that search term. In every other respect it should function just like one of the scheduled runs, but this one will only have the results from a single search, not several, like it normally does.

## Addition
Add these to the search terms:
* at the beginning 'upcoming '
* at the end 'in Brisbane or the Gold Coast'
