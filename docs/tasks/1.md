# New email feature
I want to make a new feature now where users (users MongoDB collection) can sign up for a weekly email. To start with, we'll just have a button opposite the 'Spotlight picks' header saying 'Sign up to receive weekly email'.

On clicking the button a mini-form in a pop-up window should show a form that allows a user enter their email. On validating the email, the user's email address should be saved into the database. The form then should show a thank you message.
