# Missing images
Recently a lot of the images in the list seem to be missing - even though the link seems correct, they show as broken image icons in the final generated HTML. If you look at screenshot 1 you'll see what I mean.

# Possible cause(s)
It's possible there may have been network probems, or blocking on the events website's end to prevent spammy use of assets outside of their own website?

# Idea
Once an image has been successfully downloaded should we save them by event id in a separate folder in order to prevent this happening in the future. That way the image could be used at a bigger size than the current thumbnail in the future - we could do responsive (srcset, etc.) images. 

# Investigate -> recommend
In any case, please think about what the best course of action is here, because broken links will kill interest in the site very quick!
