# LLM temperature
In some of the LLM calls, probably only the first one, where we generate the searchterms, it would be useful to be able to set the temperature of the model. Investigate if this is possible, and create a randomised temperature setting between 0 and 1 (or whatever the usual temperature variation is) so that the app gets a variety of different LLM temperature settings which I hope will give more variation to the searchterms.

Investigate if the other LLM calls the agent uses would benefit from setting a different temperature to the default.
