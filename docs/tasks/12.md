# Source of truth
There has been a mistake made in the way the HTML table is generated. It should not be generated independently of the spreadsheet - it should be based on the spreadsheet.

## Sequence of events
On each succesful run, the spreadsheet acts as the database, growing (or shrinking) over time as new queries find new events to add and out-of-date events are pruned. This spreadsheet is the app's events database, and should be thought of as the source of truth.

The HTML table, used to update the Notion page, should be generated **using the spreadheet** as its source of data. There should be an exact one-to-one correspondance between the spreadsheet and the spreadsheet.
