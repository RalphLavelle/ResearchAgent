# Updating interval type
Investigate how the interval between runs is managed, and allow as well as an hour value a minutes value. The idea is for me to be able to tighten the feedback loop and run the schedule every 5 minutes or so.

If the minutes amount is set then that's what will be used, but if it's left at 0, then the hour value will be used. Please put a comment in as well to that effect

## Example
Hours: 1, minutes: 0. Schedule runs every hour
Hours: 12, minutes: 5. Schedule runs every 5 minutes (irrelevant what the hours value is here)
