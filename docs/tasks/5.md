# Admin venues list
At the moment, the venues list shows only the information about the venues, but makes it hard to know which, or how many, actual events are associated with these venues. It would really benefit the admin user to be able to see how many gigs are on for each venue in the list, and which ones they are.

## Task
Add a column to the venues table called 'events' with the number of events associated with that venue. Make it a clickable button. WHen it's clicked, show a row underneath the venue row with the events in a list. Collapse that row on clicking again on the button.
