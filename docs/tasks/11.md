# HTML template for the data
I would like to start using Notion (again) to update a HTML table version of the spreadsheet everytime the app runs. I would like to have manual control over the HTML, so make an editable (by me) template to generate the actual HTML that will end up on Notion.

In the template, the image should show as an image in the table, and be in the first column. In the 2nd column the link should wrap around the artist's (or event's) name, not be in a separate column. The venue and location columns can be combined into one, called Venue.

Make sure the template is easy to edit, with tokens for the data to be filled in, and clear HTML markup. In fact, make the template a HTML file.
