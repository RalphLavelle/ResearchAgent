# Venues admin screen
Now to make the venues admin page functional!

## Edit
Allow an edit of each row - all it has to do is present the raw JSON document for editing, then allow that to be saved back to the database.

## Delete
Also, allow for the venue to be deleted. However, when a venue is deleted, the orphaned events that were using that venue should have their event object updated (name and id) to an existing event taken from a drop-down list that appears in the UI. Obviously, this should be not generated per-row in the HTML but can be a modular window that allows them to choose an exiting event. Then the original venue gets deleted, and the orphaned (in terms of their venue) events get updated.
