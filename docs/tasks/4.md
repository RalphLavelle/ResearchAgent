# Style upgrades
The frontend has a boring grey background - I want a nice colout palette, maybe a sofy orange, or blue, or even subtle pink - something, to make the site look more musical. Feel free to experiment

# Table/grid updates
The main events table should have each row the same height, and not have those dependent on how much text is in the About column, which looks like that's what's happening.

# Change of links
The date column should now have the link that is currently what the Event column links to. This is the actual event, as given by the date, and it should be the link to the outside web.

The event title should then change to show a nice accordion, or some modern web UI, when you click on a event name. The idea is to eventually have all the other events in the current month show in a small UI element so that someone interested in that particular band or event can drill down and see more events by the same band.

# Missing images
If there is no image, have a musical emoji show instead, filling roughly the same amount of space. The worst thing is to have empty spots in the table.

Finally - make all the rows have their content top-aligned within their cell
